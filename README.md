# ડેરી મેનેજમેન્ટ સિસ્ટમ

એક મોબાઇલ-રેસ્પોન્સિવ ડેરી મેનેજમેન્ટ સિસ્ટમ જે Django અને MySQL નો ઉપયોગ કરે છે.

## લક્ષણો

- **કસ્ટમ યુઝર મોડેલ**: મોબાઇલ નંબર સાથે લોગિન
- **પ્રાણી મેનેજમેન્ટ**: ગાય અને ભેંસનું ટ્રેકિંગ
- **પ્રેગ્નન્સી ટ્રેકિંગ**: 75 દિવસની તપાસ અલર્ટ
- **દૂધ રેકોર્ડ્સ**: ચરબી આધારિત કિંમત ગણતરી
- **મોબાઇલ-ફર્સ્ટ ડિઝાઇન**: Bootstrap 5 સાથે રેસ્પોન્સિવ UI
- **ગુજરાતી ભાષા**: બધા લેબલ અને સંદેશા ગુજરાતીમાં

## ઇન્સ્ટોલેશન

1. **રિક્વાયરમેન્ટ્સ ઇન્સ્ટોલ કરો:**
   ```bash
   pip install -r requirements.txt
   ```

2. **એનવાયરમેન્ટ સેટઅપ કરો:**
   ```bash
   cp env.example .env
   # .env ફાઇલમાં તમારા MySQL ક્રેડેન્શિયલ્સ અપડેટ કરો
   ```

3. **ડેટાબેસ સેટઅપ કરો:**
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

4. **સુપરયુઝર બનાવો:**
   ```bash
   python manage.py createsuperuser
   ```

5. **ડેવલપમેન્ટ સર્વર ચલાવો:**
   ```bash
   python manage.py runserver
   ```

## મુખ્ય ફિચર્સ

### પ્રાણી મેનેજમેન્ટ
- ટેગ નંબર, પ્રકાર (ગાય/ભેંસ), જાત
- સેવાની તારીખ રેકોર્ડ
- 75 દિવસ પછી સ્વચાલિત તપાસ અલર્ટ
- સકારાત્મક તપાસ પર અપેક્ષિત પ્રસવ તારીખ (ગાય: 280 દિવસ, ભેંસ: 310 દિવસ)

### દૂધ મેનેજમેન્ટ
- દૈનિક દૂધ રેકોર્ડ
- ચરબી પ્રમાણ અને કિંમત ટ્રેકિંગ
- સ્વચાલિત કુલ રકમ ગણતરી
- ચરબી આધારિત બોનસ ગણતરી

### ડેશબોર્ડ
- કુલ પ્રાણીઓની સંખ્યા
- આજનું કુલ દૂધ અને આવક
- તપાસ અલર્ટ્સ
- આગામી પ્રસવો
- તાજેતરના દૂધ રેકોર્ડ્સ

## ડિઝાઇન લક્ષણો

- **મોબાઇલ-ફર્સ્ટ**: બોટમ નેવિગેશન બાર
- **ડેસ્કટોપ**: સાઇડબાર નેવિગેશન
- **કલર થીમ**: ગ્રીન અને વ્હાઇટ (ક્લીન ડેરી લુક)
- **રેસ્પોન્સિવ કાર્ડ્સ**: બધા ડિવાઇસે ઓપ્ટિમાઇઝ્ડ

## ટેકનોલોજીસ

- **Backend**: Django 4.2.7
- **Database**: MySQL
- **Frontend**: Bootstrap 5, Bootstrap Icons
- **Forms**: Django Crispy Forms
- **Language**: Gujarati (gu)

## યુઝર ફ્લો

1. **નોંધણી**: મોબાઇલ નંબર સાથે એકાઉન્ટ બનાવો
2. **લોગિન**: મોબાઇલ નંબર અને પાસવર્ડથી લોગિન
3. **ડેશબોર્ડ**: મુખ્ય માહિતી અને અલર્ટ્સ જોઓ
4. **પ્રાણી મેનેજમેન્ટ**: પ્રાણી ઉમેરો અને ટ્રેક કરો
5. **દૂધ રેકોર્ડ્સ**: દૈનિક દૂધ રેકોર્ડ કરો
